When using {{< params "IDmelon" >}} Security Key as a Service, Contactless Cards can be used as a security key. The admin of your organization can activate a card for you or he can send an invitation email which help you register your card.

## Activation by admin

<hr class="hr-line">

In case of activating the card by admin, you receive an email containing a PIN number and you can use your card right away.

<p align="center">
    <img src="/images/vendor/UserPanel/active_card_1.png" class="doc-img-frame">
</p>

## Activation by user

<hr class="hr-line">
When you possess the card, admin only can send you an invitation email containing the registration link.
